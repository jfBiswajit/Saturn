# Saturn

Saturn is a online e-commerce website who provides various kind of products to customers.

## Key Features

- Payment Integration (PayPal)
- Login System
- Rating System
- User Dashboard
- Admin Panel
- Form validation

## Tecnology

- Core PHP
- MySQL
- Vania Javascript
- Ajax
- CSS
- HTML
